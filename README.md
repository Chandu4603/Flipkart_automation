# Flipkart_automation
Flipkart automation means writing a Selenium script to automatically:  
Search for a product  
Apply filters  
Extract product details (price, rating, name)  
Add items to cart  
Perform login automation  
Monitor price changes

# Selenium is an open-source tool used to automate web browsers. It allows you to:
Open websites
Click buttons
Enter text
Scrape data
Perform end-to-end testing
